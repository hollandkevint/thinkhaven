# Research References

External writing and research that informs the Thinkhaven product thesis. Each note summarizes the source and maps it to specific product decisions: positioning, mode design, target user, or roadmap.

These are intellectual foundations, not requirements. They explain *why* the product is shaped the way it is, so future contributors can argue with the premises rather than just the implementation.

## Adding a new reference

1. Create a markdown file in this folder named after the source (e.g. `author-title.md`).
2. Lead with the title, author, and link.
3. Summarize the argument in your own words.
4. Add a "Mapping to Thinkhaven" section that names specific product elements the source informs.
5. Add a row to the table above.
